# iNews

SPA web app about news in Indonesia using SPA approach

### Installation

`npm install`

#### Development

`npm start`

#### Minify and Deploy

`npm run build`

## Technology

#### Framework

[VueJS](https://vuejs.org/)

#### Libraries:

[Bootstrap](https://getbootstrap.com/)

[Axios](https://www.npmjs.com/package/axios)
